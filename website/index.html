How to use

Get productive in minutes

Follow these guides to get the most out of each major workflow in Bolt.

01
Configure companies & environments

Start by setting up your companies and environments in Settings. Everything in Bolt — from database connections to API collections — is scoped to a company + environment pair.

  • Open Settings → Companies and add each company code
  • Go to Settings → Environments and define environments (dev, staging, prod)
  • Add GCP project configs (cluster name, region, project ID)
  • Save DB secrets (host, user, password) per environment
# Settings → GCP Config
Company: acme
Environment: production
Cluster: prod-cluster-01
Region: us-central1
Project: acme-prod-infra
✓ Config saved
02
Connect to databases via jumpbox

Bolt automatically handles GCP authentication and pod discovery. Just pick a company, environment, and database — Bolt runs gcloud get-credentials and kubectl get pods behind the scenes.

  • Open Connect to Postgres from the sidebar
  • Select company and environment from the dropdowns
  • Bolt discovers the jumpbox pod automatically
  • Type your SQL and hit Run — results appear in a table
  • Save frequent queries in Saved DB Queries
# Auto-executed by Bolt
$ gcloud container clusters
  get-credentials prod-cluster-01
Fetching cluster endpoint... done.
$ kubectl get pods | grep jumpbox
jumpbox-7d6f9b-xkj2p 1/1 Running
$ kubectl exec jumpbox-... -- psql ...
✓ Query executed · 42 rows
03
Manage tasks with Kanban board

The Task Manager gives you a drag-and-drop Kanban board. Set priorities, attach files, add comments, and configure smart reminders with custom audio tones.

  • Click + New Task and fill in title, priority, due date
  • Add tags, attachments, and comments to a task
  • Drag tasks between Not Picked → In Progress → Done
  • Set reminder frequency: Once, Daily, or Custom schedule
  • Choose an audio tone (Chime, Alert, Buzz, etc.) for reminders

04
Test APIs without leaving the app

The API Collection works like a lightweight Postman. Build and save requests per collection, set headers, inspect the response body and status code — all within Bolt.

  • Open API Collection from the sidebar
  • Create a new collection and add requests (GET / POST / DELETE)
  • Set URL, headers, and body per request
  • Hit Send and view the formatted JSON response
GET /api/users?company=acme 200 OK
POST /api/orders 201 Created
DEL /api/cache/flush 404 Not Found
{
  "users": [ "alice", "bob" ],
  "total": 2
}
05
Use built-in developer utilities

Stop switching browser tabs. Bolt ships with a full suite of everyday dev tools — all offline, all instant.

  • Base64 — paste text, get encoded/decoded output instantly
  • JWT Decoder — paste a token; see header, payload & expiry in IST
  • Time Converter — live clock, convert between Epoch ↔ IST ↔ UTC
  • JSON Formatter — beautify, minify, or convert JSON ↔ CSV
