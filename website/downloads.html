⚡ Installation
1
Download the .dmg file above.
2
Open the .dmg — drag Bolt.app into your Applications folder.
3
On first launch, macOS may show a security warning. Go to System Settings → Privacy & Security and click "Open Anyway".
4
Launch Bolt from Applications. On first run, the database will be initialised automatically.
1
Download the .exe installer above.
2
Run the installer. Windows SmartScreen may warn — click "More info" → "Run anyway".
3
Follow the installation wizard. Bolt will be added to your Start Menu and Desktop.
4
Launch Bolt. The local database initialises on first run.
1
Download the .AppImage or .deb above.
2
For AppImage: chmod +x Bolt-*.AppImage then run it directly — no install needed.
3
For .deb: sudo dpkg -i bolt-*.deb to install system-wide.
4
Make sure gcloud CLI and kubectl are installed and on your PATH for GCP features to work.
